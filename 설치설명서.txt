네이버 쇼핑 광고 수집기 - 설치 설명서
=======================================

이 문서만 따라 하시면 됩니다. 순서대로 진행해주세요.


[1] Node.js 설치
-----------------
1. https://nodejs.org 접속
2. "LTS" 라고 적힌 버튼을 눌러 설치 파일 다운로드
3. 다운로드된 파일 실행 → 계속 "Next" 눌러서 기본 설정 그대로 설치
4. 설치가 끝나면 컴퓨터를 한 번 재시작해주세요 (안 하면 다음 단계에서 오류가 날 수 있습니다)

설치 확인 방법:
- 시작 메뉴에서 "cmd" 검색 → 명령 프롬프트 실행
- 아래 명령어 입력 후 엔터
    node -v
- v20.0.0 같은 버전 숫자가 나오면 정상 설치된 것입니다.


[2] Google Chrome 설치 확인
---------------------------
이 프로그램은 여러분 컴퓨터에 설치된 진짜 Chrome 브라우저를 통해 네이버에 접속합니다.
Chrome이 이미 설치되어 있다면 이 단계는 넘어가셔도 됩니다.
없다면 https://www.google.com/chrome 에서 설치해주세요.


[3] 압축 풀기
-------------
1. 전달받은 zip 파일을 원하는 위치(예: 바탕화면)에 압축 해제
2. 압축을 풀면 아래와 같은 폴더/파일이 보입니다.
    - crawler (폴더)
    - 사용설명서.md
    - 설치설명서.txt (이 파일)

주의: crawler 폴더는 이동하지 말고, 압축 푼 그 자리에서 계속 사용해주세요.
      (나중에 폴더를 옮기면 다시 설정해야 할 수 있습니다)


[4] 프로그램 준비 (최초 1회만)
------------------------------
1. 시작 메뉴에서 "cmd" 검색 → 명령 프롬프트 실행
2. 아래처럼 crawler 폴더로 이동합니다.
   (경로는 실제로 압축을 푼 위치에 맞게 바꿔주세요. 예시는 바탕화면 기준입니다)

    cd Desktop\crawler

3. 아래 명령어를 입력하고 엔터. 필요한 프로그램들을 설치하는 과정이라 1~2분 정도 걸립니다.

    npm install

   중간에 빨간 글씨(오류 메시지)가 아니라 노란 경고(warning)만 보이는 것은 정상입니다.

4. .env 라는 설정 파일이 crawler 폴더 안에 이미 들어있는지 확인해주세요.
   (탐색기에서 안 보이면 "숨김 파일 표시"를 켜야 보일 수 있습니다 - 파일 탐색기 상단 "보기" 메뉴에서 체크)

   SUPABASE_URL / SUPABASE_ANON_KEY는 이미 값이 채워져 있어 수정할 필요 없습니다.
   SUPABASE_EMAIL / SUPABASE_PASSWORD는 제가 따로 알려드린 계정 정보를 채워주세요.
   (수집 결과를 저장하려면 로그인이 필요해서 추가된 항목입니다. 메모장으로 .env 파일을 열어서
    두 줄 오른쪽에 이메일/비밀번호를 붙여넣고 저장하면 됩니다)


[5] 네이버 로그인 (최초 1회, 이후엔 자동 유지)
----------------------------------------------
1. 명령 프롬프트에서 crawler 폴더 안에 있는 상태로 아래 명령어 입력

    npm run chrome

2. 전용 Chrome 창이 새로 뜹니다. 이 창에서 본인 네이버 계정으로 로그인해주세요.
3. 로그인 정보는 저장되어서 다음부터는 다시 로그인할 필요가 없습니다.
4. 이 Chrome 창은 수집이 끝날 때까지 끄지 마세요.


[6] 실제 사용
-------------
자세한 사용법은 같이 들어있는 "사용설명서.md" 파일을 참고해주세요.
(메모장으로 열어도 되고, 더 편하게 보려면 VS Code나 Typora 같은 프로그램으로 열면 보기 좋습니다)

간단히 요약하면:
- crawler 폴더에서 "npm run dev" 입력 → 등록된 키워드 전체를 한 번 수집하고 자동 종료
  (터미널에 [CAPTCHA] 라고 뜨면, 열려있는 Chrome 창에서 캡차를 직접 풀고 명령 프롬프트로 돌아와 엔터를 누르면 이어서 진행됩니다)


[문제가 생기면]
----------------
- "npm install" 이나 "npm run ..." 명령어 자체가 안 먹는 경우 → [1]에서 Node.js 설치 후 컴퓨터 재시작을 안 했을 가능성이 높습니다.
- Chrome 연결 오류가 뜨는 경우 → [5]의 "npm run chrome"을 먼저 실행해서 Chrome 창을 띄워뒀는지 확인해주세요.
- 그 외 증상은 "사용설명서.md"의 "문제 해결" 표를 참고해주세요.
- 그래도 안 되면 저에게 화면 캡처와 함께 연락해주세요.
